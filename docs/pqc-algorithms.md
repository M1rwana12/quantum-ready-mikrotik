# PQC Алгоритмы: что использовать

| Алгоритм | Тип | Размер ключа | Рекомендация |
|--------|-----|-------------|-------------|
| **ML-KEM-768** (Kyber) | KEM | ~1184 байт | VPN, TLS |
| **ML-DSA-65** (Dilithium) | Подпись | ~2420 байт | SSH, сертификаты |
| **FN-DSA-512** (Falcon) | Подпись | ~897 байт | IoT, embedded |
| **SLH-DSA-128s** | Подпись | ~16 KB | Параноидальный режим |

**Рекомендация для MikroTik:**  
`ML-KEM-768` + `ML-DSA-65` — баланс безопасности и производительности.
